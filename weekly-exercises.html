Workshop:
Weekly Exercises

The first third of the semester will act as a workshop during which we will test technologies and learn new methods quickly via weekly exercises with the browser and code editors. In the process we will learn web programming and markup via HTML, CSS, and Javascript while also engaging in the myriad perspectives that shape online and screen-based discourse today.

The workshop / collection of exercises will be graded on:

  • Effort – Challenge yourself to do something new and unexpected. Think formally and conceptually, while also having fun.
  • On-time completion – Please post your exercises to your personal class website before the beginning of class. Late assignments will be docked one letter grade each class meeting they are late.
Exercises
  • Exercise 0. Text Editor: ASCII Town
  • Exercise 1. HTML & CSS Basics: Interview
  • Exercise 2. Arrangement: Flags
  • Exercise 3. Responsiveness: Multiples
  • Exercise 4. Animation: Sensory Experience
